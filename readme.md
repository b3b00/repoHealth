# description

outil d'analyse de la santé d'un repo github.
Il produit un ensemble de métriques heuristiques qui permettent d'établir un niveau de santé du repo.

## issues
- Calcul du temps moyen de résolution des issues. 


## releases
- calcul de la fréquence des releases.

## commitset releases
établir si un projet est toujours actifs sur la base des derniers fix d'issues et des derniers commits
